## Configuration Google OAuth

Pour activer la connexion avec Google:

1. Allez dans le dashboard Supabase
2. **Authentication** > **Providers**
3. Activez **Google**
4. Ajoutez vos credentials OAuth Google:
   - Client ID
   - Client Secret
5. Ajoutez l'URL de callback autorisée dans la console Google Cloud
